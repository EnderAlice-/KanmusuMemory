【概要】
　艦これのスクリーンショットをキッチリサイズで保存してTwitterへ投稿するための専用ブラウザです。


【注意事項】
　・艦メモは公式アプリではありません。
　・艦メモは公式の推奨するブラウザではありませんので何が起こるかわかりません。
　・艦メモを使用して起きた不具合について一切の責任をとることができません。
　・艦メモを使用して起きた不具合について運営へ問い合わせをしないでください。
　　（艦メモと完全に切り離して考えられる場合は除きます。）
　・ブラウザとしての機能は部分的なため課金などの処理を行わないでください。


【使いはじめるときの注意事項】
　・他のブラウザで出撃中であれば帰還してから使い始めてください。
　　多重ログインになって資材などが無駄になる可能性があります。
　　その他、中断されて困るようなことをしているときは必ず終わらせてからにしてください。


【セットアップ～実行】
　- Windowsの場合
　　１．任意のフォルダにアーカイブを解凍してください。
　　２．Flash Playerをインストールしてください。（普段使用しているブラウザを使用してください。）
　　　　　別のバージョンのAdobe Flash Playerをインストール
　　　　　　http://get2.adobe.com/jp/flashplayer/otherversions/
　　　　　　手順１　あなたのOSを選択
　　　　　　手順２　Flash Player 11 for Other Browsersを選択
　　３．KanmusuMemory.exeを実行してください。

　- Ubuntuの場合
　　１．任意のフォルダにアーカイブを解凍してください。
　　２．Flash Playerをインストールしてください。
　　　　UbuntuソフトウェアセンターでMozilla用を選んでください。
　　３．KanmusuMemory.shを実行してください。

　- Macの場合
　　１．dmgファイルを開いてください。
　　２．KanmusuMemoryをアプリケーションフォルダへコピーしてください。
　　３．Flash Playerをインストールしてください。（普段使用しているブラウザを使用してください。）
　　　　　別のバージョンのAdobe Flash Playerをインストール
　　　　　　http://get2.adobe.com/jp/flashplayer/otherversions/
　　　　　　手順１　Mac OS X 10.6 - 10.9
　　　　　　手順２　Flash Player 11 for Mac OS X 10.6 - 10.9
　　４．KanmusuMemoryを実行してください。


【アップデート】
　各環境とも上書きしてください。

【遊び方】
　１．起動するとDMMのログイン画面になります。
　　　IDとパスワードを入力してログインしてください。
　　　IDとパスワードを保存するかは自由です。Cookie対応してますので保存されます。
　２．艦これのゲーム画面へ移動します。
　３．「思い出を残す(F5)」を押すとスクリーンショットを撮ります。
　　　ゲーム画面が無いと失敗します。
　４．画像の保存場所を設定します。（一度だけ）
　　　初めてスクリーンショットを撮ろうとすると画像の保存場所を聞かれます。
　　　任意のフォルダを選択してください。
　５．Twitterの認証を行います。（一度だけ）
　　　自動的にブラウザを開いて認証画面へ移動します。
　　　Pinコードを入手して新たに表示されているダイアログへ入力してください。
　　「○○で、つぶやく」と画像の下に表示されればOKです。
　６．おもいっきり艦娘さんへの思いをつぶやきましょう。

　・「思い出を表示」について
　　過去に記録した画像を一覧表示します。
　　クリックすると拡大表示します。
　　一覧・拡大時に画像をダブルクリックするとつぶやくダイアログへ移動します。
　　選択ボタンでもOKです。

　・「お知らせタイマー」について
　　入渠・遠征・建造の終了時間をツイッターの自己メンションでお知らせします。
　　各時間表示の「設定」ボタンで設定モードになります。
　　時間は、時、分、秒をそれぞれ設定できます。
　　時間はテンプレートが一覧表示されますので近いものをダブルクリックでもOKです。
　　表示は、「残り時間」と「終了予定時刻」です。

　・お知らせタイマーの「音」について
　　WindowsとUbuntuは、実行ファイルと同じフォルダにある「alarm.mp3」を再生しています。
　　変更したいときは、このファイルを置き換えてください。
　　Macは、「KanmusuMemory.app/Contents/Resources/alarm.mp3」です。
