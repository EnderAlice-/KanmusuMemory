艦メモ　～艦娘メモリーズ～
===========================

艦これ専用ブラウザアプリ  

艦これのスクショをカッチリきっちりサイズで保存してTwitterへつぶやけるアプリです。  
Qtで作成しているので、Windows/Ubuntu/Mac OS Xで動作します。
